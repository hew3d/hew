# Sample-model texture assets

Wood color maps embedded into the bundled sample documents by
`build_samples.rs` (see that example's header for the build command).

Both images are downscaled 512×512 JPEG color maps from
[ambientCG](https://ambientcg.com), published under
[CC0 1.0](https://creativecommons.org/publicdomain/zero/1.0/)
(public domain — no attribution required, given anyway):

| File         | Source asset                                      |
|--------------|---------------------------------------------------|
| `oak.jpg`    | [Wood048](https://ambientcg.com/view?id=Wood048)  |
| `walnut.jpg` | [Wood062](https://ambientcg.com/view?id=Wood062)  |

Only CC0 assets may be added here — nothing that would place an
attribution or share-alike obligation on shipped Hew binaries.
