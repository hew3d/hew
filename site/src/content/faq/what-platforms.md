question: "What platforms does Hew run on?"
order: 6
---

Today, Hew runs as a web app at [app.hew3d.com](https://app.hew3d.com) in any modern, WebGL2-capable browser — Chrome, Edge, Firefox, and Safari, on macOS, Windows, or Linux. Native desktop installers for all three operating systems are in development; because the desktop build shares the same Rust-compiled-to-WebAssembly kernel and file format as the web app, they're packaging work rather than a separate rewrite.
